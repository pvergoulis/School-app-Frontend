Το Εκπαιδευτικό Πρόγραμμα

Το Coding Factory είναι ένα καινοτόμο πρόγραμμα εκπαίδευσης στην ανάπτυξη Full-Stack εφαρμογών που δίνει έμφαση στον Προγραμματισμό, το Web και τις Βάσεις Δεδομένων. Περιλαμβάνει και τις τρεις παρακάτω ενότητες:

Προγραμματισμός

Η καρδιά του Coding Factory είναι ο κώδικας. Η ανάπτυξη λογισμικού θα κυριαρχήσει στην αγορά εργασίας τα επόμενα χρόνια. Κάθε εκπαιδευόμενος θα αναπτύξει τις αναγκαίες τεχνικές δεξιότητες ώστε να μπορεί να ανταποκριθεί στις απαιτήσεις της αγοράς εργασίας.

JAVA
  • Δομημένος / Αντικειμενοστραφής Προγραμματισμός με τη γλώσσα Java
  • Προχωρημένα θέματα προγραμματισμού
  • Java ΕΕ: JSP/Servlets, Web Programming
  • Hibernate / Spring / Spring Boot / Thymeleaf
C# . NET Core
  • Αντικειμενοστραφής Προγραμματισμός με τη γλώσσα C# .NET Core
PYTHON
  • Προγραμματισμός με τη γλώσσα Python που χρησιμοποιείται ιδιαίτερα σε εφαρμογές Machine Learning και Data Science
ΜΕΘΟΔΟΛΟΓΙΕΣ / ΕΡΓΑΛΕΙΑ
  • Αλγόριθμοι και Δομές Δεδομένων
  • Αρχιτεκτονικές Ανάπτυξης Λογισμικού MVC / SOΑ / DAO / DTO
  • Συστήματα Διαχείρισης Εκδόσεων Λογισμικού Git / GitHub
  • Διαχείριση Εξαρτήσεων Λογισμικού (Dependencies) με Maven, Gradle
  • Συστήματα Ελέγχου Λογισμικού JUnit
  • Ολοκληρωμένα Περιβάλλοντας Ανάπτυξης Λογισμικού (IDEs) IntelliJ, Eclipse, Visual Studio, Visual Studio Code
  • Σχεδιασμός με UML
  • Design Patterns (Καλές πρακτικές προγραμματισμού)

Web

Η δυνατότητα των επιχειρήσεων για καινοτομία ιδιαίτερα στο Web είναι θεμελιώδης ανάγκη στη σημερινή ψηφιακή αγορά. Οι εκπαιδευόμενοι θα πρέπει να εξοικειωθούν με τα εργαλεία και της μεθοδολογίες ανάπτυξης εφαρμογών στο Web, σχεδιασμού και ελέγχου της εμπειρίας των χρηστών (UX)

JAVASCRIPT

React

TYPESCRIPT / ANGULAR
  • Προγραμματισμός Web Διεπαφών με TypeScript και Angular
UX/UI DESIGN
  • UX (User Experience)

CMS (Content Management Systems)
  • Εργαλεία Διαχείρισης Ολοκληρωμένων Εφαρμογών με Drupal

Βάσεις Δεδομένων

Οι Βάσεις Δεδομένων είναι απαραίτητες στον ψηφιακό κόσμο. Πως αποθηκεύεται η πληροφορία σε μόνιμα μέσα αποθήκευσης, αλλά και πως μπορούμε να καλλιεργήσουμε γενικές δεξιότητες που ζητούν οι επιχειρήσεις;

ΣΥΣΤΗΜΑΤΑ ΒΑΣΕΩΝ ΔΕΔΟΜΕΝΩΝ
  • Web Development με HTML/CSS
  • Προγραμματισμός Βάσεων Δεδομένων με τη γλώσσα SQL
  • Συστήματα διαχείρισης σχεσιακών Βάσεων Δεδομένων με MS SQL Server, MySQL, H2 Database
  • NoSQL Databases (Firebase / MongoDB)
ΔΙΑΣΥΝΔΕΣΗ ΒΑΣΕΩΝ ΔΕΔΟΜΕΝΩΝ
  • JDBC (Java Database Connectivity), ODBC (Open Database Connectivity)
  • Συστήματα ORM (Object-Relational Mapping)
BIG DATA
  • Διαχείριση μεγάλων δεδομένων (Big Data) με αποδοτικό τρόπο με την MongoDB
ΠΡΩΤΟΚΟΛΛΑ ΕΠΙΚΟΙΝΩΝΙΑΣ / WEB SERVICES
  • HTTP (Hypertext Transfer Protocol)
  • REST (Representational State Transfer)
  • Web Services
ΣΥΓΧΡΟΝΕΣ ΜΕΘΟΔΟΛΟΓΙΕΣ ΚΑΙ ΕΡΓΑΛΕΙΑ
  • Ολοκληρωμένα συστήματα διαχείρισης με Agile / DevOps
ΓΕΝΙΚΕΣ ΔΕΞΙΟΤΗΤΕΣ
  • Επικοινωνιακές Δεξιότητες / Διαχείριση Χρόνου / Συνεργασία / Δημιουργικότητα / Καινοτομία
